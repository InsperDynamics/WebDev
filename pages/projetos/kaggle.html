Nosso Objetivo | Idealização

       O Kaggle é uma plataforma de competições online, que apresentam diversas possibilidades de aprender programação, machine learning e outros tópicos específicos de cada competição. A abordagem que tivemos a este projeto é criar Squads de trabalho de 3-4 alunos para haver compartilhamento de conhecimento e abrangência de temas.

Como estamos?

       Hoje em 2023.1 ambos os times já realizaram todos os eventos de introdução para o Kaggle, e estão buscando desenvolver frentes de Image Recognition - que têm o objetivo em utilizar imagens 2D para chegar uma imagem 3D.
